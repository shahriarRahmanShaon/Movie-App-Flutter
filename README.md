# MovieHUB üé¨

**MovieHUB** is a Flutter-based movie application designed with an advanced MVVM architecture, providing users with a seamless experience to browse movies, add favorites, and search by title. The app supports multiple languages, customizable themes (light and dark modes), and integrates various packages to ensure an efficient and smooth user experience.

---

## üåü Features

- **Movie Dashboard**: Discover popular and trending movies on the main dashboard.
- **Add to Favorites**: Save favorite movies for easy access later.
- **Search Functionality**: Quickly find movies by title with a built-in search feature.
- **Multi-language Support**: Users can switch between multiple languages for a personalized experience.
- **Light & Dark Themes**: Toggle between light and dark modes to suit user preferences.
- **Optimized Architecture (MVVM)**: Built on a robust MVVM architecture to separate concerns and improve maintainability.
- **API Integration with Dio**: Real-time data fetched with Dio for efficient networking.
